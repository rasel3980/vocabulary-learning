# Lingo Bingo

## Purpose:
The Spanish Vocabulary Project is a web application designed to help users learn and practice Spanish vocabulary. The app offers various features such as learning new words, testing your vocabulary knowledge, and tracking your learning progress. It is built using React, Firebase, and Tailwind CSS for styling, offering a user-friendly and responsive experience.
## Live URL:
(https://vocabulary-authentication.web.app/)
## Key Feature:
- **Vocabulary Lessons**: Learn new Spanish vocabulary words, grouped by categories such as food, travel, common phrases, etc.
- **User Profile**: Create a personalized profile where you can save your learned words and track your learning progress.
- **Learning Streak**: Keep track of your daily learning streak to motivate you to continue learning.
- **Authentication**: Secure login and signup functionality via Firebase Authentication.
- **Responsive Design**: The app is fully responsive, ensuring it works smoothly on both desktop and mobile devices.
## npm Package used:
- **react**: A JavaScript library for building user interfaces.
- **react-router-dom**: For routing and navigation between pages.
- **firebase**: Firebase for handling authentication and data storage.
- **react-icons**: A library of popular icons for React applications.
- **tailwindcss**: A utility-first CSS framework for responsive design.
- **daisyUI**: A plugin for Tailwind CSS that provides pre-built UI components.
- **react-toastify**: For showing notifications and alerts within the app.
- **animate.css**: for animation card and welcome message.
